Cubic-video
## 介绍
用于摄像头数据采集上报到服务端，经过服务端处理视频后上传到文件服务器，并提供图形化界面进行数据监控和查看
主要技术有Spring Boot 、Mybatis、Netty、websocket 、MediaRecorder

## JDK版本
目前项目编译使用的是JDK1.8
